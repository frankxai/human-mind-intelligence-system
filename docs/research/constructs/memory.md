# Memory — research grounding

> Non-clinical. This grounds the memory construct model. It does not profile, score, or diagnose a person.

## The model

Memory is the encoding, storage, and retrieval of experience over time. In the ontology it sits downstream of attention (which gates it) and emotion (which modulates it), and it feeds three constructs: learning, identity, and belief. Durable change routes through memory before it reaches any of them.

## The prediction frame

A world model carries a latent state that persists across steps and is updated as new input arrives — a functional memory. The construct's classic encoding/storage/retrieval lenses sit inside that frame rather than competing with it: storage is the maintained latent, retrieval is the read-out, encoding is the update. Framing memory as a persisting, updated state keeps it honest about reconstruction — what is retrieved is the current state, not a stored snapshot.

## Anchors

- **Understanding World or Predicting Future? A Survey of World Models** — Survey, arXiv 2411.14499 (2024). Map of the world-model landscape; situates JEPA, Cosmos, and prediction-vs-generation debates. [link](https://arxiv.org/abs/2411.14499)
- **Predictive coding / the predictive brain** — Rao & Ballard; Friston; Clark, review (1999). The brain as a prediction engine minimizing prediction error across a cortical hierarchy. The cognitive-science root of the prediction frame. [link](https://en.wikipedia.org/wiki/Predictive_coding)

## Boundary

This grounds a model of how memory works. It does not license assessing a person's memory accuracy, reliability, or capacity from observed recall.
