# Behavior — research grounding

> Non-clinical. This grounds the behavior construct model. It does not profile, score, or diagnose a person.

## The model

Behavior is observable action — the construct's only directly visible output. In the ontology it is fed by motivation, decision-making, learning, belief, and identity, and it feeds back into learning. It is where committed intention becomes signal the model can read.

## The prediction frame

In active inference, action is how a system makes its predictions come true — it changes the input to fit the model rather than only updating the model to fit the input. Behavior, on this view, is the outward half of inference. Because it is the one construct that produces external signal, it is where the upstream constructs (intention, belief, motivation) become observable; the System reads that signal and stays at observation, never inferring inner state as fact.

## Anchors

- **Bayesian approaches to brain function (active inference, free energy)** — Friston et al., review (2010). Perception and action as Bayesian inference; action selected to minimize expected free energy. Grounds the motivation/behavior/belief edges. [link](https://en.wikipedia.org/wiki/Bayesian_approaches_to_brain_function)
- **An Introduction to Predictive Processing Models of Perception and Decision-Making** — Sprevak & Smith, Topics in Cognitive Science (2024). Accessible, current synthesis of predictive processing across perception and decision-making. [link](https://onlinelibrary.wiley.com/doi/10.1111/tops.12704)

## Lens alignment

Schema lenses: `habit-loop`, `dual-process`, `behavioral-economics`, `reinforcement-learning`. The active-inference anchor informs `reinforcement-learning` (action selected over expected outcomes) and the prediction-driven reading of `dual-process`; the predictive-processing synthesis backs the same. `habit-loop` and `behavioral-economics` come from the construct module.

## Boundary

This grounds a model of how action is produced. It does not license inferring a person's hidden intentions or character from a single observed behavior.
