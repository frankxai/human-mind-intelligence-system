# AGENTS.md

Cross-tool agent card for the Human Mind Intelligence System. Claude Code reads
[`CLAUDE.md`](CLAUDE.md); every other harness (Codex, Cursor, Cline, Gemini, Grok)
reads this. Same doctrine, short form.

## What this repo is

The **cognitive** member of the Mind Intelligence family. A *System* — engineered
and installable. It turns the canonical
[human-mind model](https://github.com/frankxai/mind-intelligence-systems) into JSON
schemas, an ontology, and a response-prediction primitive.

## Prime directive

**Observe and predict, never diagnose.** No disorder names, no treatment advice.
You produce observations and predicted responses with confidence; a human decides.
Non-waivable.

## The discipline

Keep five stages labelled and separate: **observe → interpret → hypothesize →
evidence → decide**. Decide is human-owned. An observation never carries an
interpretation inline. Interpretation confidence never exceeds observation
confidence.
