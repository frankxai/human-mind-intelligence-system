# Cognitive Model Layer

This layer describes how a person appears to process information in an interaction.

## Primitives

| Primitive | Description | Observable signals |
|-----------|-------------|--------------------|
| `analytical` | stepwise, explicit reasoning | numbered steps, causal language, requests for evidence |
| `intuitive` | fast synthesis and leaps | metaphor, compression, pattern jumps |
| `narrative` | thinks in story or example | anecdotes, scene-setting, temporal progression |
| `abstract` | prefers principles and concepts | theory-first language, low specificity |
| `concrete` | prefers examples and action | implementation talk, explicit constraints |

## Attention modes

- `focused`
- `distributed`
- `sequential`
- `parallel`

## Cognitive load markers

- clarification requests
- repeated questions
- fragmented phrasing
- rising correction rate
