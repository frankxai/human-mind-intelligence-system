# Emotional Model Layer

This layer models emotional interaction patterns that are visible in language and behavior. It is observational and non-clinical.

## Patterns

| Pattern | Description | Observable signals |
|---------|-------------|--------------------|
| `engaged` | energized and participatory | elaboration, follow-up questions, expressive language |
| `withdrawn` | low-energy or low-disclosure | short replies, reduced detail |
| `frustrated` | blocked, tense, or resistant | repetition, sharp framing, escalation cues |
| `curious` | exploratory and open | hypotheses, what-if questions |
| `confident` | declarative and certain | low hedging, direct claims |
| `uncertain` | hesitant or provisional | hedging, qualifiers, requests for reassurance |
| `collaborative` | seeks shared construction | inclusive language, coordination phrases |
| `guarded` | protective and careful | evasive framing, low disclosure |

## Guardrail

These are not diagnoses, syndromes, or mental health conclusions.
