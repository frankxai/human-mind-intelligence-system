# Social Model Layer

This layer describes relational stance and communication style within an interaction.

## Relational primitives

| Primitive | Description | Observable signals |
|-----------|-------------|--------------------|
| `high_trust` | open and assumes good faith | disclosure, warm coordination |
| `low_trust` | guarded or verification-seeking | requests proof, formality, skepticism |
| `authority_oriented` | sensitive to hierarchy | title references, deference or challenge |
| `peer_oriented` | favors horizontal relation | informality, equal framing |
| `independent` | self-directed stance | low reassurance seeking |
| `interdependent` | consensus-oriented stance | questions, alignment checks, inclusive framing |

## Communication style measures

- `directness`
- `formality`
- `verbosity`
- `questioning_rate`
